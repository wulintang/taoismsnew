# TAOISCMS

道教之家CMS开源项目（TAOISCMS），是由南通伍林堂文化传播有限公司旗下的伍林堂CMS开源项目部、盘龙阁权益保护委员会、伍林堂表砸开源项目部、 伍林堂科技事业发展部联合开发的收费服务型CMS内容管理系统。系统采用前后端分离技术，灵活云数据库管理。有多种优点：第一是它可以为宫观、协会、社会团体提供页面生成病管理；第二是它可以结合其他第三方的CMS作为插件使用。
